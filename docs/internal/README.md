# Internal Docs

`docs/internal/tun-host.md` 用來紀錄 TUN / WireGuard 主機的實際登入資訊與操作備忘，但該檔案已加入 `.gitignore`，不會隨版本控制分發。

建議內容範例：

```
# TUN Host Notes (local-only)
- 用途：...
- 連線方式：...
- 工作目錄 / 備份流程：...
- git push 限制 / 其他安全措施：...
- 變更紀錄：...
```

> 注意：任何含有密碼、金鑰或伺服器細節的文件都應放在 `.gitignore` 列出的檔案中，或改存至密鑰管理系統，避免誤入 repo。
